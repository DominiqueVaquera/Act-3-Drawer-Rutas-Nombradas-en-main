# Prompt
- Lenguaje flutter nivel principiante.
- El negocio es un zoologico.
- Drawer ( con 4 opciones).
  encabezado: imagen avatar del negocio(https://raw.githubusercontent.com/DominiqueVaquera/Imagenes/refs/heads/main/c2.jfif), desde la red, nombre empresa (Zoologico), dirección (Av. Avena), teléfono(6567863680), correo(zoologico@gmail.com),
- Listile: icono relacionado al zoologico, texto y acción para abrir pagina según opción de listile, (Visitantes, empleados, Animales...)
- En cada pagina una imagen centrada de 200x200 desde la red  (desde github)
- Utilizar rutas nombradas para navegar entre paginas desde el drawer (rutas nombradas).
- Ten en cuenta que todas la rutas deben ir en la carpeta "LasPaginas" y hazmelo segun los diseños de las imagenes.
- Recuerda usar el color verde atractivo y tenue en todo excepto en letras, en las letras si tienen fondo verde deben ir blanco pero si tienen fondo blanco deben ir negras.
